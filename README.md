`cp` is a `color-picker` tools

> the color template is [weilanwl/ColorUI](https://github.com/weilanwl/ColorUI)

### Developer

use `Reactjs`

```console

git clone https://github.com/luxizhizhong/color-picker cp
cd cp

yarn

# run server
yarn start

# build static page
yarn build

```
